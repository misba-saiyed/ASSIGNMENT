  4. An HTML entity is a piece of text ("string") that begins with an ampersand ( & ) and ends with a semicolon ( ; ). Entities are frequently used to display reserved characters which would otherwise be interpreted as HTML code, and invisible characters like non-breaking spaces. Generally, you don't need to use HTML entities if your editor supports Unicode. For some instances, entities can be useful: Your editor does not support Unicode. Your keyboard does not support the character you would like to type, such as em-dash or the copyright symbol.

  12. To make a hyperlink in an HTML page, use the anchor start tag and anchor end tags, which are the tags used to define the links. The anchor tag indicates where the hyperlink starts and the anchor end tag indicates where it ends. Whatever text gets added inside these tags, will work as a hyperlink.
